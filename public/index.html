Welcome to the Ticketing API

Use the following endpoints to interact with the API:

  • POST /api/tickets/reset - Reset All Tickets
  • POST /api/tickets - Create a new ticket
  • PATCH /api/tickets/:seatNumber - Update a ticket status
  • GET /api/tickets/status/open - View all open tickets
  • GET /api/tickets/status/closed - View all closed tickets
  • GET /api/tickets/:seatNumber - View Ticket Status by Seat Number
  • GET /api/tickets/user-details/:seatNumber - View User Details by Seat Number
